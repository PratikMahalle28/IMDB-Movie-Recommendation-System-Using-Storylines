# IMDB-Movie-Recommendation-System-Using-Storylines
IMDb Movie Recommendation System Content-based recommender using TF-IDF + cosine similarity on 5000+ IMDb 2024 storylines. Interactive Streamlit app. Paste any plot → get top similar movies! NLP - scikit-learn - pandas pip install -r requirements.txt &amp;&amp; streamlit run app.py Skills: Text preprocessing, vectorization, ML similarity.
